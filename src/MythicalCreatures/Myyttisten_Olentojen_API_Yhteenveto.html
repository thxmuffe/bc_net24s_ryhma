Myyttisten Olentojen Web API - Yhteenveto

Tämä projekti on **verkkopalvelu**, joka tallentaa ja hallinnoi myyttisiä olentoja. Sen sijaan, että käytettäisiin tietokantaa, olennot tallennetaan **JSON-tiedostoon** (tekstitiedosto, joka sisältää tietoa jäsennellyssä muodossa). Projekti toimii **.NET 9 Web API**:n avulla, ja Visual Studio Code on asetettu niin, että se on helppo käynnistää, ajaa ja testata.

🛠 Vaiheet API:n rakentamiseen

1️⃣ Luo projekti

dotnet new webapi -n MythicalCreatures
cd MythicalCreatures
code .

Poista WeatherForecast.cs-tiedosto (sitä ei tarvita).

2️⃣ Helpota projektin ajamista

Aseta **VS Code** niin, että voit käynnistää API:n **F5-näppäimellä** tai oikotien avulla.

3️⃣ Määritä verkkopalvelu (Program.cs)

Tämä tiedosto **käynnistää palvelimen** ja varmistaa, että API voi käsitellä pyyntöjä.

4️⃣ Määrittele, mikä on olento (Models/Creature.cs)

  • Id (yksilöllinen numero)
  • Name (olennon nimi)
  • Type (tuli, vesi, ilma jne.)
  • Strength, Agility, Intelligence (ominaisuudet)

5️⃣ Tallenna olennot tiedostoon (Data/CreatureRepository.cs)

Olennot tallennetaan **JSON-tiedostoon**. Koodi voi:

  • **Ladata** olennot tiedostosta
  • **Tallentaa** muutokset, kun olentoja lisätään, muokataan tai poistetaan

6️⃣ Luo ohjain (Controllers/CreaturesController.cs)

Käyttäjät voivat:

  • **Hakea kaikki olennot** (GET /creatures)
  • **Hakea yhden olennon** (GET /creatures/1)
  • **Lisätä uuden olennon** (POST /creatures)
  • **Päivittää olennon** (PUT /creatures/1)
  • **Poistaa olennon** (DELETE /creatures/1)

7️⃣ Käynnistä API

  • Avaa **Run & Debug** -paneeli VS Code:ssa.
  • Paina **F5** tai käytä oikotietäsi.

8️⃣ Testaa API

**Luo testitiedosto** (test.http) ja käytä **VS Code REST Client** -laajennusta testaukseen.

🔍 Mitä tarkistaa, jos API ei toimi

🔴 API ei käynnisty?

  • Varmista, että Program.cs sisältää AddControllers().
  • Katso, että **VS Code asetukset** rakentavat projektin ennen ajamista.

🔴 Olennot eivät tallennu?

  • Onko creatures.json olemassa?
  • Onko SaveCreatures()-metodia kutsuttu?

🔴 API ei vastaa?

  • Sisältyykö MapControllers() Program.cs-tiedostoon?
  • Kokeile käynnistää API uudelleen.

✅ Lopullinen tarkistuslista ennen ajamista

  • 🔹 **Rakenna projekti** (dotnet build tai **F5 VS Code:ssa**)
  • 🔹 **Käynnistä API** (**F5** tai oikotie)
  • 🔹 **Testaa API** käyttämällä test.http-tiedostoa tai Postmania
